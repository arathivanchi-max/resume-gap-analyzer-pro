# Resume Gap Analyzer Pro

An AI-powered career strategy assistant that helps job seekers analyze role fit, identify resume gaps, improve ATS alignment, and prepare for interviews.

---

# Overview

Resume Gap Analyzer Pro acts as a recruiter, resume strategist, and interview coach in one system.

The GPT compares a candidate’s:
- Resume
- LinkedIn profile
- Portfolio or GitHub projects
- Career notes
- Supporting materials

against a target job description to:
- Identify skill and experience gaps
- Improve ATS optimization
- Rewrite resumes strategically
- Surface transferable skills
- Prepare users for interviews
- Strengthen positioning for competitive roles

The system is designed for:
- Software engineers
- Technical professionals
- Product and platform engineers
- Career transition candidates
- Return-to-work professionals
- Mid-level to senior candidates

---

# Core Features

## Gap Analysis

Compares candidate materials against a target role and identifies:
- Technical skill gaps
- ATS keyword gaps
- Experience mismatches
- Seniority alignment issues
- Missing domain exposure

Outputs include:
- Match percentage
- Critical gaps
- Minor gaps
- Hiring-risk indicators

---

## Transferable Skills Mapping

Maps existing experience to adjacent or related requirements.

Examples:
- Backend engineering → platform engineering
- Data systems → observability
- Infrastructure work → distributed systems
- Automation projects → AI tooling workflows

Provides:
- Resume reframing guidance
- Positioning recommendations
- Honest assessment of unbridgeable gaps

---

## ATS-Optimized Resume Rewriting

Generates:
- Tailored professional summaries
- Optimized bullet points
- ATS keyword alignment
- Role-specific positioning

Rules:
- Never fabricate experience
- Preserve candidate voice and tone
- Prioritize measurable outcomes
- Keep formatting ATS-friendly

---

## Gap Mitigation Strategy

Suggests:
- Certifications
- Portfolio projects
- LinkedIn improvements
- GitHub enhancements
- Short-term learning plans

Useful for:
- Career pivots
- Re-entering the workforce
- Expanding into adjacent technical domains

---

## Interview Preparation

Generates:
- Likely interview questions

- Technical positioning guidance

Focuses on:

Interactive interview simulation with:

# Philosophy

Strong resumes do not invent experience.

They:
- clarify impact,
- surface transferable strengths,
- align with hiring signals,

This GPT is built around that principle.
